# STREAMLIT Examples 
## example1.py 
- streamlist에서 메뉴 간에 데이터 공유 예제 
- 실행 시키면 첫번째 메뉴에서 사이드바가 표시되고 0~10 까지 값을 선택 한후 메뉴2를 선택하면 메뉴1에서 선택된 값이 메뉴2에 표시 됨. 
- 메뉴2에서 나온 사이드바에서 값을 바꾸면 똑같이 메뉴 1에 적용됨
